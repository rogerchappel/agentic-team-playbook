# Code Of Conduct

## Standard

This project is for practical engineering collaboration.

Be direct, respectful, and specific. Critique ideas and changes, not people.

## Expected Behaviour

- assume good intent
- keep feedback actionable
- explain tradeoffs clearly
- respect maintainers' decisions
- avoid personal attacks, harassment, or discriminatory language

## Unacceptable Behaviour

- harassment or abuse
- threats or intimidation
- discriminatory comments
- publishing private information
- sustained off-topic disruption

## Enforcement

Maintainers may remove comments, close issues, block users, or decline contributions that violate this code of conduct.

For concerns, contact the project maintainer through the GitHub repository.
